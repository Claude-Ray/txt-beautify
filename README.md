# txt-beautify
Beautify text files for better reading experience on kindle.

## Features
- Without calibre
- No need to convert into mobi

主要用于kindle传书，不需要calibre，不需要转mobi，因为亚马逊会自动转换txt文件。

这里仅修饰文件内容，以便亚马逊转出格式更优美的文本。

现在只是一个简单粗暴的脚本，要求文件编码utf-8，满足基本需求。
